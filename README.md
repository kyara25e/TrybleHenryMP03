<h2 align="center">🧠 Project Overview</h2>

<p>
This project is a <b>full stack web application</b> that allows users to interact with dynamic data through a modern interface.
It simulates a real production environment, integrating frontend and backend with a clean architecture.
</p>

<ul>
  <li>Client–server architecture</li>
  <li>API integration</li>
  <li>Dynamic rendering</li>
  <li>Scalable structure</li>
</ul>


<h2 align="center">⚡ Key Features</h2>

<ul>
  <li>🔐 Authentication system (Register / Login)</li>
  <li>🔄 REST API communication</li>
  <li>📦 Dynamic data rendering</li>
  <li>🔎 Filtering and search</li>

<h2 align="center">🚀 Getting Started</h2>

<h3>1. Clone the repository</h3>

<pre>
https://github.com/kyara25e/TrybleHenryMP03.git
</pre>

<h3>2. Install dependencies</h3>

<pre>
cd client
npm install

cd ../server
npm install
</pre>

<h3>3. Run the project</h3>

<pre>
• frontend
npm run dev

• backend
npm start
</pre>


<h2 align="center">🔐 Environment Variables</h2>

<pre>
API_URL=your_backend_url
DB_URL=your_database_url
JWT_SECRET=your_secret
</pre>
